## âš¡ Quick Start

### Prerequisites
- Node.js v18+
- Python 3.10+ (for Logic Engine)

### Installation
```bash
# Clone the repository
git clone https://github.com/vm799/Maestro.git

# Navigate to root
cd Maestro

# Install Dependencies
npm install

# Run the Development Server
npm run dev
```

### Configuration
Update `antigravity.config.json` to set your governance policies:
```json
{
  "compliance_standards": ["ISO 42001", "EU AI Act"],
  "risk_tolerance": "Low"
}
```
